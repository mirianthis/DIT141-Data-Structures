
# HuffmanProject

This code is part of a project for the Data Structures course at Harokopio
University of Athens, Dept. of Informatics and Telematics. It implements a 
code that reads from the file frequencies.dat and writes a huffmantree in a new 
file named tree.dat.

## Usage

Compile using 

```
mvn compile
```

Create a jar using 

```
mvn package
```

Run main using 

```
java -cp target/HuffmanProject_2-1.0-SNAPSHOT org.hua.huffmanProject_2.Main

```
