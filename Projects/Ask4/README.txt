
# HuffmanProject

This code is part of a project for the Data Structures course at Harokopio
University of Athens, Dept. of Informatics and Telematics. It implements a 
code that reads a file and writes to another file the encoded version of it .

## Usage

Compile using 

```
mvn compile
```

Create a jar using 

```
mvn package
```

Run main using 

```
java -cp target/HuffmanProject_4-1.0-SNAPSHOT org.hua.huffmanproject_4.Main

```
