
# HuffmanProject

This code is part of a project for the Data Structures course at Harokopio
University of Athens, Dept. of Informatics and Telematics. It implements a 
code that reads 3 files and finds writes in a txt file the frequencies of 
certain characters

## Usage

Compile using 

```
mvn compile
```

Create a jar using 

```
mvn package
```

Run main using 

```
java -cp target/HuffmanProject-1.0-SNAPSHOT org.hua.HuffmanProject.Main

```
