
# HuffmanProject

This code is part of a project for the Data Structures course at Harokopio
University of Athens, Dept. of Informatics and Telematics. It implements a 
code that reads an encoded file and writes to another file the decoded version of it .

## Usage

Compile using 

```
mvn compile
```

Create a jar using 

```
mvn package
```

Run main using 

```
java -cp target/HuffmanProject_5-1.0-SNAPSHOT org.hua.huffmanproject_5.Main

```
